# Todo List App with React and Node.js

**Difficulty:** Junior  
**Deadline:** 3 days

## Scenario

As a React developer, you are tasked with creating a simple Todo List application that allows users to add, remove, and mark tasks as completed. The application should have a user-friendly interface and interact with a Node.js backend to store and retrieve tasks. The application should be able to handle multiple users and their respective tasks.

## Requirements

1. Create a React application with a form to add new tasks and a list to display existing tasks
2. Implement a Node.js backend to store and retrieve tasks from a database or in-memory storage
3. Implement functionality to mark tasks as completed and remove tasks from the list
4. Ensure the application is user-friendly and responsive
